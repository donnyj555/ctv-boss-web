# How to get your API Keys

## OpenAI API Key (For Scripts & Voiceovers)

1.  Go to **[platform.openai.com/api-keys](https://platform.openai.com/api-keys)**.
2.  If prompted, log in with your normal ChatGPT/OpenAI account.
3.  On the left sidebar, click **API keys**.
4.  Click the green button that says **+ Create new secret key**.
5.  In the popup:
    *   **Name:** Type something like "CTV Boss Ad Creator" so you remember what it's for.
    *   **Project:** Leave as default.
    *   **Permissions:** Leave as "All".
6.  Click **Create secret key**.
7.  **IMPORTANT:** A popup will appear showing your key (it starts with `sk-...`). **Copy this key immediately and save it somewhere safe on your computer (like a notepad file).** You will not be able to view it again once you click "Done".

*Note: OpenAI requires a payment method on file to use their API, as it costs a few fractions of a cent per generation. If you haven't added billing details, go to **Settings > Billing** to add a minimum balance (usually $5 is plenty for months of testing).*

## Creatomate API Key (For Video Rendering)

1.  Go to **[creatomate.com](https://creatomate.com/)** and click "Sign Up" or "Start for free".
2.  Create an account (you can use Google login or email).
3.  Once logged into the Creatomate dashboard, look at the bottom left of the sidebar and click on **Project Settings** (it looks like a gear icon).
4.  Scroll down to the **API Key** section.
5.  You will see a long string of letters and numbers. Click the "Copy" icon next to it to copy your API key.
6.  Save this key in the same safe place you put your OpenAI key.

*(Creatomate provides a free sandbox mode, so you don't need a credit card just to build and test the integration!)*
